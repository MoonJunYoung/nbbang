### 시작 가이드
---
**Requirements**
<br/>
<br/>
for building and running the application you need
 - Node.js
 - Npm
<br/>

**Installation**
<br/>
```bash
$ git clone https://github.com/moonjunyoung/nbbang.git
$ cd frontend
```

**Frontend**
<br/>
```bash
$ cd frontend
$ npm install
```
